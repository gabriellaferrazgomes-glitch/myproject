# Análise de Veículos

Este projeto é um aplicativo web desenvolvido com Streamlit.

## Funcionalidades
- Visualização de histograma da quilometragem dos veículos
- Botão interativo para gerar o gráfico

## Tecnologias
- Python
- Pandas
- Plotly
- Streamlit
